# 🐍 Snake Game (Java Swing)

A classic Snake Game built using Java Swing with smooth controls, dynamic speed, scoring system, and interactive UI features.

---

## 🎮 Features

* 🟢 Snake movement using arrow keys
* 🍎 Food (apple) generation at random positions
* ⚡ Speed increases as snake eats food
* 🧮 Live Score system
* 🔁 Restart game (ENTER key / button)
* ⏸ Pause/Resume using SPACE key
* 🎨 Custom UI (background, colors, grid, etc.)
* 🌈 Dynamic snake color based on size

---

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* AWT (Graphics, Events)

---

## ▶️ How to Run

1. Clone the repository

   ```
   git clone https://github.com/your-username/snake-game.git
   ```

2. Open in IntelliJ / Eclipse

3. Run the `Snake.java` file

---

## 🎯 Controls

* ⬅️ ➡️ ⬆️ ⬇️ : Move snake
* SPACE : Pause / Resume
* ENTER : Restart game
